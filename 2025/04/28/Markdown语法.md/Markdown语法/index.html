Markdown语法

ctrl + B 加粗

ctrl + i 斜体

三个-或者*建立分割线

下划线的使用和html中类似，在需要添加下划线的文字首尾添加<u>文本</u>​

两个`之间输入文字，当第二个被输入就会自动形成灰色框

高级用法：
